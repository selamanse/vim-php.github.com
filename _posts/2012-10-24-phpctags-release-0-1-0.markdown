An enhanced [ctags](http://packagist.org/packages/techlivezheng/phpctags) compatible index generator written in pure PHP. Released under GPLv2 license. 
The release 0.1.0 is ready to go!

## Changelog

* ctags compatible
* surppoted tokens
    * constant
    * variable
    * function
    * class
    * class method
    * class property
    * class constant
    * interface
* scope field support
* access field support
